# PRODIGY_WD_01
Responsive Landing Page with Interactive Navigation Menu

This project is a responsive landing page built using HTML, CSS, and JavaScript. It features a fixed interactive navigation menu that changes style when the user scrolls through the page or hovers over menu items. The design is fully responsive and works smoothly across different screen sizes and devices.

Features

* Responsive landing page design
* Fixed navigation bar visible on all sections
* Navigation menu changes color/style on scroll
* Hover effects on menu items
* Smooth and interactive user experience
* Built with pure HTML, CSS, and JavaScript

Technologies Used

* HTML5
* CSS3
* JavaScript

Project Objective

The main objective of this project is to create an attractive and user-friendly landing page with an interactive navigation menu that enhances the overall browsing experience.

How It Works

* The navigation bar remains fixed at the top of the page.
* When the page is scrolled, the menu style changes dynamically using JavaScript.
* Hovering over menu items highlights them with visual effects.
* Responsive CSS ensures proper layout on mobile, tablet, and desktop devices.

Learning Outcomes

Through this project, I learned:

* Responsive web design
* DOM manipulation using JavaScript
* CSS hover and transition effects
* Creating fixed navigation menus
* Improving website interactivity and UI design
